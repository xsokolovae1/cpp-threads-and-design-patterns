# C++ Threads and Design Patterns

A collection of C++ exercises completed as part of university coursework, covering multithreading and object-oriented design patterns.

## Contents

### `threads/`
Exercises on concurrent programming using the C++ Standard Library:
- `std::thread`, `std::mutex`, `std::condition_variable`
- Producer-consumer synchronization with `std::queue`

### `oop-patterns/`
Implementations of classic OOP design patterns in C++:
- Strategy
- Decorator
- Factory
- Singleton
- Adapter
- Observer
- Builder
- Facade
- Command
- State
